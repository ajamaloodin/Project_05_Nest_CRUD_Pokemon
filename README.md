# Execute in Dev Environment
1. Clone the repository
2. Execute the following command:
```
    yarn install
```
 3. You should have the NEST CLI installed:
 ```
    npm i -g @nestjs/cli
```
 4. Start up the DB:
 ```
    docker-compose up -d
```

## Stack:
- Nest
- MongoDB
